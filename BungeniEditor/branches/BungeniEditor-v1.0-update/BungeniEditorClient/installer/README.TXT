BungeniEditor requires :

OpenOffice.org 3.0 +
Java 1.6 +

on Windows XP or Ubuntu 8.04
on Ubuntu you will need to uninstall the OpenOffice.org installation shipped
with Ubuntu and install the debian package from http://download.openoffice.org

The editor source code can be accessed from :

http://bungeni-editor.googlecode.com/svn/BungeniEditor/trunk

The google-code project for the website is at :

http://code.google.com/p/bungeni-editor

The project wiki is at :

http://code.google.com/p/bungeni-editor/wiki

The project mailing list is at :

http://groups.google.com/group/akomantoso-dev


Bungeni.org is an Open Source parliamentary Information System.
